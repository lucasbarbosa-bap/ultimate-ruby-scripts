<h1> Ultimate Ruby tests ⭐ </h1>
<p> Repositório criado para armazenar meu primeiro contrato e testes com uma nova linguagem <strong> Ruby </strong> >_< <hr /> 
Tenho 3 scripts simples até agora:
</p>
<details>
    <summary>lobinha.rb</summary>
    <blockquote>
        <p> Script simples de terminal para me familizar com a sintaxe e flow da linguagem <br /> Um RPG simples no universo de Honkai: star rail com foco na personagem SilverWolf. </p>
    </blockquote>
</details>

<details>
    <summary>ibuki.rb</summary>
    <blockquote>
        <p> Teste de interface gráfica (GUI) e da biblioteca "Gosu", esta sendo focada em game dev. <br /> Interface simples renderizando apenas um botão, efeito sonoro e uma imagem (sprite), esta sendo condicional e dentro seu proprio sistema de aleatoriedade. <br /> Tematizado com a Ibuki Mioda de Danganronpa 2. </p>
    </blockquote>
</details>

<details> 
    <summary>monopad.rb</summary>
    <blockquote>
        <p> Um CLI que fiz para relembrar os conceitos de classes e entender como o ruby trata e lida com manipulação de arrays e dados. <br /> Há um menu, onde podemos ver a lista de alunos, adicionar um aluno ou executar um aluno (puhuhu). <br /> Uma especie de "Lista de Chamada Escolar", inpirada totalmente em Danganronpa 2 com as personagens do elenco como mock.</p>
    </blockquote>
</details>

<details> 
    <summary>mikan.rb</summary>
    <blockquote>
        <p> API simples da Mikan Tsumiki para entender como o ruby funciona o back end em ruby. <br /> Um protótipo simples usando o framework Sinatra para criar uma API com um mock básico. <br /> O objetivo foi entender roteamento (GET), parâmetros de URL e renderização em tela. </p>
    </blockquote>
</details>

<h2> Como rodar? </h2>

<blockquote>
<h3> Instalação: </h3>
<p> git clone https://github.com/lucasbarbosa-bap/ultimate-ruby-scripts.git</p>
<p> cd RUBY </p>
<h4> Bibliotecas: </h4>
    <p> <strong style="font-style:italic"> Lobinha </strong> <br /> Não precisa de nada, puro Ruby!</p>
    <p> <strong style="font-style:italic">Ibuki: </strong> <br /> gem install gosu </p>
    <p> <strong style="font-style:italic"> Monopad: </strong> <br />gem install tty-prompt colorize </p>
    <p> <strong style="font-style:italic"> Mikan: </strong> <br />gem install sinatra sinatra-contrib </p>
</blockquote>

<blockquote>
<h3> Runing: </h3>
<p> ruby (nome_do_arquivo) </p>
<p> Ou use a extensão <strong style="font-style:italic"> Code Runner </strong> </p>
</blockquote>
